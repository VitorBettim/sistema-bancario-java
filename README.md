# Sistema Bancário em Java (POO)

Projeto desenvolvido para praticar Programação Orientada a Objetos em Java.

## Funcionalidades

- Criar contas bancárias
- Depositar dinheiro
- Sacar dinheiro
- Transferir entre contas
- Consultar saldo

## Conceitos aplicados

- Programação Orientada a Objetos
- Encapsulamento
- Classes e Objetos
- Construtores
- Métodos
- ArrayList

## Estrutura do projeto

Banco.java → Gerencia as contas do banco  
Conta.java → Representa uma conta bancária  
Cliente.java → Representa o cliente do banco  
Main.java → Execução do sistema  

## Tecnologias

- Java
